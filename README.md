# PAUSE

**PAUSE** is a mental wellness app that analyzes journal entries to predict users' moods and recommend personalized games designed to support emotional well-being. By combining natural language processing (NLP) with psychological insights, PAUSE provides an engaging and secure space for self-reflection and stress management.

## Features

- **Mood Prediction**: Uses sentiment analysis to assess the user's emotional state from journal entries.
- **Personalized Game Recommendations**: Suggests interactive games tailored to support different mental health needs.
- **Journaling Assistance**: Encourages consistent writing habits with prompts and insights.
- **Privacy-Focused**: Ensures journal entries remain secure and confidential.

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/atharvakokane/UAE-InnovAIte-2025.git
   ```
2. Navigate to the project directory:
   ```sh
   cd UAE-InnovAIte-2025
   ```
4. Run the application:
   ```sh
   python mainProgram.py
   ```

## Usage

1. Open the app and log in.
2. Write a journal entry.
3. View your mood analysis.
4. Play a recommended game based on your emotional state.
